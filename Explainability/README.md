# Explainable Machine Learning Demonstration

This repository demonstrates **interpretable machine learning techniques** using the classic **Iris dataset**. The notebook shows how to train a machine learning model and interpret its predictions using **feature importance**, **LIME**, and **SHAP**.

---

## Contents

- `explainability.ipynb` – The main notebook containing:
  - Loading and preprocessing the dataset
  - Training a Random Forest classifier
  - Visualizing **feature importance**
  - Applying **LIME** for local explanations
  - Applying **SHAP** for global and local explanations
- `iris.csv` – The Iris dataset in CSV format (for easy use)
